# 图片缩放工具 (Image Scaler)

一个简单的命令行工具，用于按指定比例等比例缩放图片（JPG/PNG）。

## 功能

- 加载 JPG/PNG 图片
- 按指定比例（0.1-1）等比例缩放
- 保持图片质量
- 详细的步骤日志输出
- 支持自动在输入文件同目录生成缩放版本

## 安装依赖

```bash
pip install -r requirements.txt
```

## 使用方法

```bash
python main.py --input <输入文件路径> --scale <缩放比例> [--output <输出目录路径>]
```

### 参数说明

- `--input`: 输入图片文件路径（JPG/PNG，必须）
- `--scale`: 缩放比例，必须是 0.1 到 1 之间的小数（必须）
- `--output`: （可选）输出目录路径，缩放后的图片将保存到此目录（文件名与输入相同）。如果未提供，则在输入文件同目录生成 `{原文件名}_scaled.{扩展名}`

### 示例

```bash
# 将图片缩放为原始大小的 50%，输出到指定目录
python main.py --input ./photo.jpg --scale 0.5 --output ./resized

# 将图片缩放为原始大小的 80%，输出到指定目录
python main.py --input /path/to/image.jpg --scale 0.8 --output /path/to/output

# 缩放为原始大小的 30%，自动在同目录生成 photo_scaled.jpg
python main.py --input photo.jpg --scale 0.3

# 缩放 PNG 图片，自动在同目录生成 image_scaled.png
python main.py --input image.png --scale 0.7

# 缩放 JPG 图片，指定输出目录
python main.py --input input.jpg --scale 0.5 --output ./output_dir
```

## 输出

- 如果提供了 `--output` 参数：缩放后的图片将保存在指定的输出目录中，文件名与输入文件相同
- 如果未提供 `--output` 参数：在输入文件同目录生成 `{原文件名}_scaled.{扩展名}` 文件
- 如果输出目录不存在，程序会自动创建
- 程序会输出详细的步骤日志，格式为 "1/7: 步骤描述"

## 错误处理

- 如果输入文件不存在或不是 JPG/PNG 格式，程序会显示错误信息
- 如果缩放比例不在 0.1-1 范围内，程序会显示错误信息
- 如果图片加载或保存失败，程序会显示详细的错误信息

## 注意事项

- 支持 JPG/JPEG/PNG 格式的图片
- 缩放比例必须介于 0.1 和 1 之间（包含）
- 输出目录如果已存在，不会清空其中的文件
- 如果输出目录中已存在同名文件，会被覆盖
- 当未指定 `--output` 时，生成的文件名为 `{原文件名}_scaled.{扩展名}`，如果该文件已存在，会被覆盖
